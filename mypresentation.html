Vamos começar?

Como fazer um Deck?

  1. Escreva os slides:

    O conteúdo de um slide é simplesmente HTML.

  2. Escolha os temas:

    Um para o estilo dos slides e outro para as  transições do deck.

  3. Inclua extensões:

    Adicione funcionalidade extra ao seu deck, ou deixe-o simples.

Início Rápido:

Quando você fizer o download do deck.js, inclua um arquivo chamada boilerplate.html. Você pode ir editando imediamante seus slides desta página e visualizá-los em seu navagedor. Depois, ao sentir-se confortável customizando o deck, você poderá editar diversar partes do boilerplate ou criar um que atenda às suas necessidades.

A marcação:

Slides são apenas elementos HTML com a classe css slide.

        
        <section class="slide">
          <h2>How to Make a Deck</h2>
            <ol>
              <li>
              <h3>Write Slides</h3>
              <p>Slide content is simple HTML.</p>
              </li>
              <li>
              <h3>Choose Themes</h3>
              <p>One for slide styles and one for deck transitions.</p>
              </li>
              …
          </ol>
        </section>

Estilos de Temas:

Customize as cores, tipografia e layout do  conteúdo do slide.

<link rel="stylesheet" href="/path/to/css/style-theme.css">

Temas de Transição:

Defina transições entre slides usando transições do CSS3. Navegadores antigos farão fallback, mas você não está utilizando esses navegadores em suas apresentações, não é ?…

<link rel="stylesheet" href="/path/to/css/transition-theme.css">

Extensões:

O Core fornece a funcionalidade básica de um slide com navegação através das setas esquerda e direita, mas você pode querer mais. Aqui estão as que estão incluídas neste  deck:

  • deck.goto: Adiciona um atalho para pular para qualquer slide numerado. Tecle g, digite o número do slide e tecle t enter.
  • deck.menu: Adiciona um menu de visualização, permitindo ver todos os slides em uma grid. Tecle m para sair do menu de visualização, continue navegando em seu deck e tecle m para retornar à visualização normal. Dispositivos touch pode dar toque duplo no deck para mudar entre as  visualizações.
  • deck.navigation: Adidiona botões clicáveis (esquerda/direita) para aqueles que não gostam do  teclado.
  • deck.status: Adiciona um indicador do número da página. (atual/total)
  • deck.scale: Dimensiona cada slide para ajustar-se ao container do deck usando CSS Transforms nos navegadores que dão suporte.

Cada pasta de extensão no pacote de download contém os arquivos JavaScript, CSS e HTML necessários. Para a lista completa de módulos de extensão incluídos no deck.js, veja a documentação.

Slides Aninhados:

O último slide contém alguns passos para criar subpassos em slides. Basta aninhá-los:

        
                <section class="slide">
                  <h2>Extensions</h2>
                  <p>Core gives you basic slide functionality...</p>
                  <ul>
                    <li class="slide">
                      <h3>deck.goto</h3>
                      <p>Adds a shortcut key to jump to any slide number...</p>
                    </li>
                    <li class="slide">...</li>
                    <li class="slide">...</li>
                    <li class="slide">...</li>
                  </ul>
                </section>
